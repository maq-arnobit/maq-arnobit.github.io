Java String Methods

Here are some useful methods for manipulating Strings in Java.


length()

Returns the length of the string sequence.

Output:

subtring(startIndex), substring(startIndex,endIndex)

Returns a portion of the string denoted by start and end indices. The end index is exclusive.

Output:

For the first example, the substring starts at index 3 of the string and continues until the end of the string.

For the second example, the substring starts at index 3 of the string and ends at index 5, but index 5 is exclusive so only index 3 and 4 are shown.

toUpperCase, toLowerCase

Returns a string with all characters in uppercase or lowercase

Output:

charAt(index)

Returns the character from the specified position on the string
