How to: Make a Simple Guessing Game in Java

The rules for this game are as follows:

1. The program will generate a number with two possible values, 1 and 0

2. The user will input their guess

3. If guessed correctly, the user's score increases, and the program will repeat step 1 and 2

4. If guessed wrong, the user's score is shown at the end and the program stops.


Create a new Java file.

In the main method, create a Random object. This will be used to generate random numbers.

Create a Scanner object. This is needed to get the user input.

Next, create an integer variable that will hold the player's score, and a boolean variable that will be checked by the loop.

Create a while loop with the condition "!done". This means that the while loop will continuously execute until the boolean variable done becomes true.

In the loop, generate a random integer using random.nextInt(2). This returns a value between 0 and 1, as 2 is exclusive.

Prompt the user to input their guess. Store the value in a new integer variable.
